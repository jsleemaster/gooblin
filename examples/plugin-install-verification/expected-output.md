# Expected Output

## Gooblin Council

### The Clipper

Cut the unsupported claim. Keep only the install paths that have been checked.

### Ground Control

The distribution goal is plugin-first and skill-pack-compatible, not universal marketplace approval.

### Rubber Duck

Separate facts:

- Which CLI commands exist?
- Which local installs succeeded?
- Which remote installs were checked after release?
- Which host-specific claims are still unknown?

### Yak Shaver

Defer broad compatibility language until each host is tested.

## Verdict

Do this: document verified local CLI versions and install results.

Do not do this: claim official support, universal compatibility, or marketplace approval.

Verify with: installer output, plugin list status, and a release note that names the exact environment checked.
