# Continue-Style Context Adapter

Use this recipe for agents that let you add repository files as persistent context.

## Suggested Context

Add these paths:

- `AGENTS.md`
- `skills/gooblin/SKILL.md`
- `skills/_core/safety-floor.md`
- `commands/shipcheck.md`
- `evals/rubric.md`

## Activation Prompt

```text
Use Gooblin council mode for this task. Keep the change small, safe, reproduced, scoped, and verified.
```

## Status

This is a manual context recipe. It does not require accounts, API keys, telemetry, cloud services, or a central server.
