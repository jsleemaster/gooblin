# Cursor Adapter

Gooblin includes a Cursor-style fallback rule:

```text
.cursor/rules/gooblin.mdc
```

Use it when Cursor-style project rules are available.

## Setup

Keep the rule in `.cursor/rules/` and keep the source behavior in:

- `AGENTS.md`
- `skills/`
- `commands/`

## Maintenance

If Gooblin behavior changes, update the rule only as a short pointer. Do not duplicate full skill bodies into the rule.

## Status

This is a readable adapter recipe, not a verified marketplace integration claim.
