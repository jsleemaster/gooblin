# /clip

## Purpose

Advanced direct-call shortcut for The Clipper. Most users should start with `/gooblin`.

## When To Use

Use this when a change may be too large, dependency-heavy, rewrite-heavy, or abstraction-heavy.

## Language

Respond in the user's language by default. Keep code, commands, file paths, API names, and error text unchanged unless translation is requested.

## Output Shape

```markdown
## The Clipper

### Cut
### Reuse
### Native option
### Smallest safe change
### Verification
```

## Example Invocation

```text
/clip This patch adds a new dependency and helper layer. What can be cut or reused?
```
