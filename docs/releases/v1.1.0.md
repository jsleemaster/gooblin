# Gooblin v1.1.0 Release Review

## Goal

Ship the router-first plugin update so users can start with `/gooblin` instead of knowing which teammate to call.

## Changes

- Repositioned `/gooblin` as the primary router interface.
- Added task-type routing rules for The Clipper, Ground Control, Rubber Duck, Yak Shaver, Shipcheck, and full council.
- Marked character commands as advanced direct-call shortcuts.
- Updated README, install docs, command docs, skill docs, adapters, hooks, examples, and eval fixtures to match router-first behavior.
- Removed direct comparison language from visible copy.
- Replaced the README date-picker Before/After example with a settings-change example.
- Updated plugin metadata to `1.1.0`.
