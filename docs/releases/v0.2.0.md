# Gooblin v0.2.0 Release Review

## Goal

Verify the plugin-first distribution path without turning Gooblin into an app, runtime, or dependency-heavy tool.

## Changes

- Added Codex and Claude marketplace manifests.
- Added a thin `plugins/gooblin` marketplace adapter that points back to the root skill pack.
- Fixed Claude plugin metadata validation.
- Fixed hook event names and hook manifest shape for Claude Code.
- Added command frontmatter for slash-command discovery.
- Added compatibility notes with actual local installer verification.
- Added two examples: unsafe auth shortcut and plugin install verification.
