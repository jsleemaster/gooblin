# Gooblin v0.3.0 Release Review

## Goal

Add optional adapters, safer hook modes, and eval fixtures while keeping Gooblin readable as plain Markdown.

## Changes

- Added `adapters/` recipes for generic agents, Cursor-style rules, and Continue-style context.
- Added `GOOBLIN_HOOK_MODE` support with `quiet`, `brief`, `verbose`, and `off`.
- Added fixture-based behavior checks under `evals/fixtures/`.
- Linked adapters and fixtures from README and docs.

## Verification

- JSON manifest parse check.
- Hook smoke tests for default, quiet, verbose, and disabled modes.
- `claude plugin validate .`
- `codex plugin list` for installed local v0.3.0 candidate after reinstall.
- `git diff --check`
- Skill frontmatter check.

## Review Notes

Adapters are recipes, not official support claims.

Fixtures are behavior checks, not benchmarks.
