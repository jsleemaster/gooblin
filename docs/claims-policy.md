# Claims Policy

Gooblin should stay honest.

## Allowed Claims

You may claim:

- Gooblin is an open-source plugin-style Agent Skills pack.
- Gooblin remains usable as readable Markdown.
- Local install paths were verified when the exact command, host version, date, and result are documented.
- Hooks are optional and reminder-only.

## Disallowed Claims

Do not claim:

- Official support from a host vendor unless that vendor explicitly says so.
- Marketplace approval unless the marketplace has actually approved it.
- Adoption numbers.
- Benchmarks.
- Security, safety, speed, cost, or code-size improvements without a documented method and dataset.
- Integrations that have not been tested.

## Benchmark Requirements

If benchmarks are added later, include:

- Method.
- Repo and task set.
- Model and agent version.
- Sample size.
- Limitations.
- Reproduction instructions.
- Raw results or enough detail to audit the claim.
